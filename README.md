## <span id="instalacion"></span> 💻 Instalación Local

Para visualizar y trabajar en este proyecto localmente, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/JFrivasIngCode/programa-ya.git](https://github.com/JFrivasIngCode/programa-ya.git)
    ```

2.  **Acceder a la carpeta del proyecto:**
    ```bash
    cd programa-ya
    ```

3.  **Ejecutar el proyecto:**
    No requiere dependencias externas. Simplemente abre el archivo `index.html` en tu navegador o utiliza la extensión **Live Server** de VSCode.

## <span id="tecnologias"></span> 🛠️ Tecnologías Utilizadas

El proyecto fue construido utilizando estándares modernos de desarrollo web:

* **HTML5:** Estructura semántica para SEO y accesibilidad.
* **CSS3:** Uso de **Custom Properties (Variables)** para un rebranding instantáneo, Flexbox y CSS Grid.

* **Unsplash:** Imágenes de alta resolución para el entorno visual.

## <span id="uso"></span> 📖 Uso del Código

### Personalización de Estilos
El proyecto utiliza variables globales en CSS para facilitar cambios de marca rápidos. Solo necesitas editar el archivo CSS principal:

```css
:root {
    --primary: #fbc02d; /* Color principal de Programa YA! */
    --dark: #1a1a1a;    /* Fondos y textos oscuros */
    --light: #f4f4f4;   /* Fondos secundarios */
    --white: #ffffff;
}

```
